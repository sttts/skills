# sttts-skills

Claude Code skills for personal workflow.

## Installation

```bash
/plugin marketplace add sttts/skills
/plugin install beads
/plugin install worktree
```

## Skills

### beads

Issue tracking workflow integration using [beads](https://github.com/sttts/beads).

Triggers on: "bd", "beads", "what's next", "add task", "add epic"

Requires: `.beads` directory in project root

### worktree

Git worktree management guidance.

Triggers on: "worktree", "worktrees", "create worktree"
